# AppGen Frontend

Visual schema builder and code generator for FastAPI applications.

## Features

- Visual schema builder with drag-and-drop nodes
- Model and relationship editor
- Automatic foreign key and bidirectional relationship creation
- Export schemas to FastAPI-compatible format

## Tech Stack

- React 18 + TypeScript
- Vite
- React Flow (schema visualization)
- Zustand (state management)
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000
```
